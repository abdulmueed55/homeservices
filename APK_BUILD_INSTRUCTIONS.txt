HomeServ APK Build Instructions

This environment does not include Android SDK/Gradle build tools, so the APK must be built in Android Studio.

FASTEST METHOD:
1. Extract HomeServ_Android_Kotlin_XML.zip.
2. Open Android Studio.
3. File > Open > select the HomeServ folder.
4. Let Gradle sync finish.
5. Go to Build > Build Bundle(s) / APK(s) > Build APK(s).
6. APK output will be here:
   HomeServ/app/build/outputs/apk/debug/app-debug.apk
7. Transfer app-debug.apk to your Android phone and install it.

If Android shows “install blocked”, enable:
Settings > Security/Privacy > Install unknown apps > allow for your file manager/browser.
