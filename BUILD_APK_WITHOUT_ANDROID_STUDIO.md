# Build APK without Android Studio

Use GitHub Actions. You only need a browser and a GitHub account.

## Steps

1. Open GitHub and create a new repository named `HomeServ`.
2. Upload all project files/folders from this `HomeServ` folder into the repository.
   - Make sure `.github/workflows/build-apk.yml` is also uploaded.
3. Open your repository on GitHub.
4. Go to the **Actions** tab.
5. Select **Build HomeServ Debug APK**.
6. Click **Run workflow**.
7. After the workflow finishes, open the completed run.
8. Scroll down to **Artifacts**.
9. Download **HomeServ-debug-apk**.
10. Extract it; inside you will find `app-debug.apk`.
11. Send `app-debug.apk` to your Android phone and install it.

## APK file generated by GitHub

```text
app/build/outputs/apk/debug/app-debug.apk
```

## Note

This creates a debug APK, which is fine for testing on your own mobile. For Play Store upload, you need a signed release APK/AAB.
